# 웹 개발 기초

HTML, CSS, JavaScript를 활용한 웹 개발 입문 강의입니다.

## 커리큘럼

1. **웹 개발 소개** - 웹의 동작 원리와 개발 환경 설정
2. **HTML 기초** - 시맨틱 태그, 폼, 테이블
3. **CSS 스타일링** - 선택자, 박스 모델, Flexbox, Grid
4. **JavaScript 기초** - 변수, 함수, DOM 조작
5. **실전 프로젝트** - 반응형 포트폴리오 사이트 만들기

## 사용법

이 리포지토리를 온라인강의실 플랫폼에 연결하면 자동으로 강의 콘텐츠가 동기화됩니다.
